# 💪 FitFusion AI  
### AI-Powered Fitness Tracking & Workout Planning Platform  

## 📌 Overview  
FitFusion AI is a full-stack fitness platform that integrates AI-based pose estimation, workout planning, and performance analytics into a single application.  
It uses computer vision to track exercises, count repetitions, and analyze posture using a webcam, eliminating the need for wearable devices.

---

## 🚀 Features  
- User authentication and profile management  
- Exercise library and workout planner  
- Real-time pose detection using webcam  
- Automatic repetition counting  
- Workout session tracking  
- Analytics dashboard for progress monitoring  

---

## 🧠 Tech Stack  
Frontend: React, HTML, CSS, JavaScript  
Backend: FastAPI (Python), REST APIs, JWT Authentication  
Database: SQLite  
Machine Learning: MediaPipe Pose  

---

## 🏗️ System Architecture  
User → Frontend → Webcam Input → Pose Estimation → Rep Counting → Backend → Database → Dashboard  

---

## ⚙️ Installation & Setup  

Prerequisites: Node.js, Python 3.8+, Git  

Clone Repository:  
git clone [https://github.com/your-username/fitfusion-ai.git]
cd fitfusion-ai  

Frontend Setup:  
cd frontend  
npm install  
npm start  

Backend Setup:  
cd backend  
pip install -r requirements.txt  
uvicorn main:app --reload  

---

## ▶️ Usage  
1. Register or login
